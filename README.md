# Knowledge Base

Zentrale Sammelstelle für **alle Skills und das Wissen** von
`fmea.generic.solutions@gmail.com`.

Dieses Repo ist gleichzeitig:

- 📓 ein **Obsidian-Vault** – einfach diesen Ordner in Obsidian als Vault öffnen
  (`.obsidian/` ist bereits vorkonfiguriert: Vorlagen, Graph, Backlinks).
- 🤖 ein **Claude-Code-Skills-Repo** – Skills unter `.claude/skills/` werden von Claude
  in jeder Session in diesem Repo automatisch erkannt und genutzt.

Beide Welten liegen konfliktfrei nebeneinander.

## Konventionen

- **Sprache:** Deutsch.
- **Verlinkung:** Obsidian-Wikilinks `[[Notiz]]` statt loser Aufzählungen, damit der
  Graph wächst und Themen verbunden bleiben.
- **Dateinamen:** sprechend, mit Bindestrich oder Leerzeichen; ein Thema pro Notiz.
- **Neue Notiz:** mit [`Vorlagen/Notiz-Vorlage.md`](Vorlagen/Notiz-Vorlage.md) starten.
- **Neuer Skill:** mit [`Vorlagen/Skill-Vorlage.md`](Vorlagen/Skill-Vorlage.md) starten und
  unter `.claude/skills/<name>/SKILL.md` ablegen.
